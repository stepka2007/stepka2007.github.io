Быстрое начало работы

Новое

Начать работать с системой можно без ведения товарно-складского учета (без учета остатков продуктов и товаров на складе), только регистрировать продажи. Для этого можно пропустить пункты 9-11.

Вход в панель управления

Административная панель служит для управления программным обеспечением FUSION POS.

  • 1. Перейдите по ссылке из письма ***.fusion24.ru, где *** - название вашего домена, полученного при регистрации
  • 2. В поле логин укажите данные из письма
  • 3. В поле пароль данные из письма
  • 4. Нажмите кнопку войти
